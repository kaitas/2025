# MkDocs Material Tips

完全なドキュメントは[mkdocs.org](http://mkdocs.org)にあります.

## コマンド

* `mkdocs new [dir-name]` - 新しいプロジェクトを作成.
* `mkdocs serve` - 自動でリロードするサーバを起動.
* `mkdocs build` - サイトをビルド.
* `mkdocs help` - ヘルプを表示.

## レイアウト

    mkdocs.yml    # 設定ファイル
    docs/
        index.md  # ホームページ
        ...       # それ以外のページ
